# Business_Insight_360
**Business Insights 360** is a dynamic **Power BI analytics project** built to uncover deep business insights for **AtliQ Hardware**. The dashboard integrates data across **Finance**, **Sales**, **Marketing**, **Supply Chain**, and **Executive Operations**, empowering stakeholders with real-time visibility and data-driven decision-making for sustainable growth.

# Key Features
- **Finance View**: Analyze **Profit & Loss statements**, **product-wise performance**, and **net sales trends** to evaluate overall financial health.
- **Sales View**: Explore **customer** and **product performance** using **interactive charts** and **dynamic filters** for detailed insights.
- **Marketing View**: Examine **regional performance**, **campaign effectiveness**, and **product metrics** to assess marketing impact.
- **Supply Chain View**: Monitor **forecast accuracy**, **net error metrics**, and **inventory performance** to enhance operational efficiency.
- **Executive View**: Access **high-level KPIs**, **revenue breakdowns**, and **strategic insights** for data-driven decision-making.

# Data Sources
The dashboard consolidates data from both **Excel/CSV files** and a **MySQL database**.
While Excel and CSV files supply **target and market share information**, the MySQL database serves as the **central source for fact and dimension tables**, delivering a unified and reliable dataset for business analysis.
